Sample Page for ChrysaLisp Browser

About This Page

This is a sample HTML file loaded via file:// protocol.

Features Demonstrated

  • File I/O operations
  • HTML parsing from filesystem
  • DOM tree construction
  • Page metadata extraction

Browser Capabilities

The ChrysaLisp browser can:

  • Load HTML files from disk
  • Parse HTML into DOM structure
  • Extract page metadata (title, etc.)
  • Render DOM tree as formatted output

This demonstrates the file:// protocol support for local file access.
