This is a demonstration of the ChrysaLisp HTML browser, which was ported from KDE's KHTML rendering engine.

Technology Stack

  • Parser: Custom HTML tokenizer and parser
  • DOM: Document Object Model with Element, Text, Comment nodes
  • CSS: Stylesheet parser with selector matching
  • Renderer: Canvas-based graphical renderer
  • Interactivity: Clickable hyperlink detection
