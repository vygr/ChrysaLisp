Features

This browser supports:

  • HTML parsing and DOM construction
  • CSS styling with selectors
  • Graphical rendering to canvas
  • Clickable hyperlinks (local files only)
  • LispScript execution (ChrysaLisp instead of JavaScript!)
  • onclick inline event handlers
  • DOM manipulation from scripts
  • Text selection and clipboard integration
  • Back/Forward navigation with history
