### (mail-forget key)

### (mail-mbox) -> netid

### (mail-read mbox) -> :nil | msg

## VP methods

### :alloc -> sys/mail/alloc

```code
inputs
:r0 = mail size (bytes)
outputs
:r0 = mail message (ptr)
:r1 = string data (pubyte)
trashes
:r0-:r6
```

### :alloc_mbox -> sys/mail/alloc_mbox

```code
outputs
:r0 = mailbox id (uint)
:r1 = mailbox address (ptr)
trashes
:r0-:r5
```

### :alloc_obj -> sys/mail/alloc_obj

```code
inputs

trashes
:r0-:r5
```
